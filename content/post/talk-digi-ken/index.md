当社の矢崎裕一が、一般社団法人ユースデモクラシー推進機構が主催する「デジタル憲法フォーラム（デジ憲）」に登壇いたしました。

<!--more-->


* **イベントテーマ:** 「デジタルネイティブと考えるデジタル時代の憲法と地域の未来」
* **当社の関与:** オブザーバートークセッションに登壇
* **トークセッションテーマ:** 「私たちの思い描く未来社会」
* **登壇者名義:** Code for Tokyo 代表／「RESAS の教科書」共同執筆者
* **概要:** 憲法記念日を前に、デジタル時代における憲法のあり方や地域の未来について、若者・現役世代を中心に議論するフォーラムに、有識者の一人として参加いたしました。
* **会場:** デジタルハリウッド大学院 駿河台ホール
